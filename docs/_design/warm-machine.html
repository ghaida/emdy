The Reading Problem

Markdown is everywhere now. A good reader isn't.

AI tools generate Markdown constantly: project plans, documentation, research summaries, meeting notes. You're reading more Markdown than ever, but every app you have wants you to write it.

Emdy is different. It's a reader, not an editor. Open a folder, browse your files, and read them rendered the way they were meant to look. That's it.

What it does well

Four things, done right

Beautiful rendering

Full GFM support with syntax-highlighted code blocks, tables, task lists, and footnotes. Typography that does the content justice.

Sidebar navigation

Open any folder and browse all its Markdown files. Command palette search across filenames and content. Find what you need fast.

Zoom and fonts

Three font families to choose from: sans, serif, and mono. Zoom in and out freely. Make the reading experience yours.

Export and share

Copy as formatted HTML for pasting into any app. Export to PDF for sharing. Print with proper page breaks. Get content where it needs to go.

Philosophy
"Less, but better."
Dieter Rams

Most software tries to do everything. Emdy does one thing: it reads Markdown files, and it does that well. There is no editor, no plugins, no configuration maze. Just a calm, well-crafted reading experience.

We believe that reading deserves the same thoughtfulness as writing. The typefaces are carefully chosen. The spacing is deliberate. The interface stays out of your way so you can stay with the words.
