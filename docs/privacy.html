Privacy Policy

Effective April 2, 2026

Summary

Emdy does not collect, transmit, or store any personal data. Your files stay on your device.

What Emdy does not do

  • Emdy does not collect analytics or usage data.
  • Emdy does not send crash reports.
  • Emdy does not track which files you open, how often you use the app, or any other behavior.
  • Emdy does not use cookies, fingerprinting, or any other tracking technology.

Data stored locally

Emdy stores the following data on your device only, in the application support directory:

  • Display settings (font, theme, zoom level) — saved to a local JSON file so your preferences persist between sessions.
  • Usage counters (number of files opened, number of app launches, first launch date) — used solely to determine when to show a support prompt within the app. This data is never transmitted anywhere.

You can delete this data at any time by removing the Emdy application support folder.

Network activity

The following network activity may occur:

  • Update check. On each launch, Emdy fetches a small file from emdyapp.com to check if a newer version is available. This request contains no personal data, no device identifiers, and no usage information — it is a simple file download.
  • Remote images in Markdown files. If a Markdown file you open references images by URL, Emdy will fetch those images to display them. This is initiated by the content you choose to open, not by Emdy itself.
  • External links. When you click a link within Emdy, it opens the URL in your default web browser. Emdy does not track or intercept these navigations.

Payments

Emdy uses Stripe to process optional pay-what-you-want contributions. If you choose to make a payment, you are interacting with Stripe directly. Stripe's privacy policy governs how your payment information is handled. Emdy does not receive, store, or have access to your payment details.

Children's privacy

Emdy does not knowingly collect any information from anyone, including children under 13.

Changes to this policy

If this policy changes, the updated version will be posted here and the effective date will be updated. Since Emdy collects no data, meaningful changes to this policy are unlikely.

Contact

Questions about this policy can be directed to emdy@ghaidazahran.com.
