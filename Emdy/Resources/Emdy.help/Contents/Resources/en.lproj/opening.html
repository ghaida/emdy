Emdy Help

Opening files

Open a single file

Choose File > Open (⌘O) and select a Markdown file. You can also double-click any .md file in Finder if Emdy is set as the default Markdown reader.

Browse a directory

Choose File > Open and select a folder instead of a file. Emdy shows a sidebar listing all Markdown files in that directory, including subfolders. Click any file in the sidebar to read it.

Drag and drop

Drag a Markdown file onto the Emdy icon in the Dock, or drop it into an open Emdy window.

Open Recent

Choose File > Open Recent to reopen a recently viewed file.
