Emdy Help

Display settings

Zoom

Adjust the document display size:

  • ⌘+ — Zoom in
  • ⌘− — Zoom out
  • ⌘0 — Reset to actual size

Font style

Switch between three bundled font families:

  • ⌘1 — Sans-Serif
  • ⌘2 — Serif
  • ⌘3 — Monospace

Theme

Choose a color theme for the app:

  • ⌘7 — Light
  • ⌘8 — Dark
  • ⌘9 — System (follows macOS appearance)
