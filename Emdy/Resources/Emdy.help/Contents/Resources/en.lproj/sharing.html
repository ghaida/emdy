Sharing and exporting

Copy as RTF

Select text in the document and press ⌘C. The formatted text is copied to the clipboard as RTF, so pasting into apps like Pages, Mail, or Notes preserves headings, bold, links, and other formatting.

Export as PDF

Press ⇧⌘E or choose File > Export PDF. A save dialog lets you choose the destination. The PDF preserves the document layout and current theme.

Print

Press ⌘P or choose File > Print to open the standard macOS print dialog.
